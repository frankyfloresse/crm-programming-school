# CRM Programming School API

NestJS backend API for managing programming school CRM system.

## Quick Start

```bash
npm install
npm run start:dev
```

API runs on `http://localhost:3001`  
Swagger docs: `http://localhost:3001/api`

## Main Scripts

- `npm run start:dev` - Development server
- `npm run migration:run` - Setup database
- `npm run seed` - Add sample data

## Environment Setup

`cp .env.example .env`
